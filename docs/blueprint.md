# **App Name**: QubixPrep

## Core Features:

- Authentication & Role-Based Access: Secure user registration and login with persistent sessions and distinct workflows for Student and Admin roles.
- Student Dashboard: A centralized hub featuring daily progress trackers, 'continue studying' shortcuts, and AI-driven content recommendations.
- Admin Dashboard: A robust management interface for administrators to upload PDFs and videos, organize subjects/units, and monitor platform-wide user analytics.
- Interactive Notes System: Advanced PDF viewer with support for highlighting, sticky notes, bookmarking, and automatic last-page memory.
- AI Note Summarizer Tool: An AI-powered tool that decides which sections of a medical note are most critical to include in a concise summary.
- AI Quiz & Flashcard Generator Tool: An intelligent tool that uses reasoning to generate relevant MCQs and flashcards from specific study topics.
- Global Fuzzy Search: An instant search system across all notes, videos, quizzes, and flashcards with support for partial matches.
- Subject & Topic Architecture: Hierarchical organization of medical content into Subjects, Units, and Topics with progress tracking.
- Study Tracking & Heatmaps: Visual representation of study habits using heatmaps, daily streaks, and productivity scores.
- AI Mind Map Generator Tool: A visualization tool that constructs interactive mind maps by identifying core medical concepts and their relationships.

## Style Guidelines:

- Primary: Academic Indigo-Violet (#704DCC) for key interactions and brand identity.
- Background: Deep Purplish-Grey (#18141E) for a premium, low-strain dark mode experience.
- Accent: Electric Cyan/Blue (#7FB5FF) for highlights and interactive feedback.
- Primary Font: 'Inter' (sans-serif) for high legibility in dense academic text.
- Consistent use of Lucide line icons for a clean, modern medical-professional aesthetic.
- Mobile-first with a bottom navigation bar; Desktop utilizes a permanent sidebar. Uses glassmorphism for depth.
- Smooth transitions and micro-interactions powered by Framer Motion for a fluid feel.
